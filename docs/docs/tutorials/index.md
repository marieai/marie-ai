# Tutorials

Step-by-step tutorials for building document processing solutions with Marie-AI.

## Available tutorials

| Tutorial | Description | Time |
| -------- | ----------- | ---- |
| [Invoice processing](./invoice-processing.md) | Extract data from invoices: line items, totals, vendor info | 30 min |
| [Form extraction](./form-extraction.md) | Extract fields, checkboxes, and tables from forms | 30 min |

## What you'll learn

These tutorials guide you through building production-ready document processing pipelines:

- **Document types**: Define input and output schemas
- **Executors**: Build specialized processing components
- **Flows**: Orchestrate multi-step pipelines
- **Clients**: Send documents for processing
- **Patterns**: Field detection, validation, schema mapping

## Prerequisites

Before starting, ensure you have:

- Marie-AI installed (`pip install marie-ai`)
- Python 3.10+
- Completed the [Quickstart](../getting-started/quickstart.md)

## Choosing a tutorial

**Start with Invoice processing if you:**
- Work with financial documents
- Need to extract structured tabular data
- Want to learn OCR integration patterns

**Start with Form extraction if you:**
- Process application forms or surveys
- Need checkbox and field detection
- Want to map data to custom schemas
